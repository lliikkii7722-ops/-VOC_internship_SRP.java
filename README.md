# Student Result Portal - Java


## Description
Java console project developed during VaultofCodes internship. It stores student data, calculates total marks, percentage, and grade, and displays results using OOP concepts and ArrayList.

## Features
- Add multiple students
- Automatic calculation of total, percentage, and grade
- Dynamic storage using ArrayList
- Simple console interface

## Technologies Used
- Java
- OOP Concepts: Class, Object, Constructor, Method
- ArrayList
- Arrays
- Scanner for input

## How to Run
1. Clone the repository
2. Open project in Eclipse or any Java IDE
3. Run `VOC_internship_SRP.java`
4. Enter student details as prompted
